### Installing

- Server:

  - Write on the termianl `cd ~ && cd server && npm install`

- Web:

  - Write on the termianl `cd ~ && cd web && npm install`

- Mobile:
  - Write on the termianl `cd ~ && cd mobile && npm install`

- Server:

- Web:

- Mobile:
  - Write on the termianl `cd ~ && cd mobile && npm start`
